# 🏫 匿名授業レビュー共有アプリ

## 📝 プロジェクト概要

本プロジェクトは、学生が匿名で授業の口コミや評価を共有できるWebアプリケーションを開発するものです。ユーザーはログイン後、自由に授業を登録し、星評価付きのレビューを投稿できます。人気のある授業の検索や、詳細な口コミの閲覧を通じて、履修選択の助けとなることを目指します。

主要な開発言語として **PHP** を使用し、データベース連携を含めたアプリケーション全体を構築します。

## ✨ 主な機能（Features）

### ユーザー認証機能 (`auth` ディレクトリ担当)

| 機能 | 詳細 |
| :--- | :--- |
| **ログイン** | メールアドレスとパスワードによる認証フォームを設置し、ログイン処理を実行。セッション管理によりログイン状態を保持する。 |
| **ログアウト** | ログイン状態を解除する。 |
| **アカウント登録** | （可能であれば）ユーザーが新規にアカウントを登録できる機能。 |

### 授業・コンテンツ管理機能

| 機能 | 詳細 | 担当ディレクトリ |
| :--- | :--- | :--- |
| **授業登録** | 授業名と教授名の入力フォームを設置し、`courses` テーブルにデータを追加する。 | `class` |
| **口コミ投稿** | 授業の選択（ドロップダウン/検索）、星評価（5段階などを想定）、口コミ本文の入力フォームを設置し、`reviews` テーブルにデータを追加する。 | `review` |

### 閲覧・検索機能

| 機能 | 詳細 | 担当ディレクトリ |
| :--- | :--- | :--- |
| **検索/掲示板** | 授業名、先生名、関連用語での検索フォームを設置。 | `search` |
| **人気ランキング** | 口コミ件数が多い授業順にソートして一覧表示する機能。 | `search` |
| **授業詳細** | 特定の授業に関する全ての口コミを表示するページ。口コミは新着順などで並べ替えることができる。 | `class` |

## 🛠️ 技術スタック（Technology Stack）

| 分野 | 技術/言語 | 備考 |
| :--- | :--- | :--- |
| **メイン言語** | PHP | アプリケーションロジックの中核 |
| **フロントエンド** | HTML, CSS, JavaScript | ユーザーインターフェースの構築 |
| **データベース** | （未定） | 授業情報 (`courses`)、レビュー (`reviews`)、ユーザー情報 (`users`) などを格納 |
| **バージョン管理** | Git / GitHub | チーム開発のハブとして利用 |

## 👥 チーム体制と役割分担

プロジェクトの円滑な進行のため、ディレクトリ単位で担当者を割り振ります。

| 担当ディレクトリ | 担当者（学年） | 主な機能範囲 |
| :--- | :--- | :--- |
| **`auth`** | 1年生 | ログイン、ログアウト、セッション管理、ユーザーDB連携 |
| **`class`** | 2年生以上 | 授業登録ページ、授業詳細ページ（口コミ一覧表示） |
| **`review`** | 2年生以上 | 口コミ投稿ページ（授業選択、評価、レビューDB連携） |
| **`search`** | 2年生以上 | 検索ページ/掲示板、人気授業ランキング |
